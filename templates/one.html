Include in HTML

You can either download the files directly on your machine, or use CDN links.
(For this demonstration we've opted to use CDN links, these are on your index.html page.)
		
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/shepherd/1.3.0/css/shepherd-theme-arrows.css" />
<script src="https://cdnjs.cloudflare.com/ajax/libs/tether/1.1.1/js/tether.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/shepherd/1.3.0/js/shepherd.min.js"></script>

Initializing Shepherd

Start using Shepherd

var tour;
tour = new Shepherd.tour({
	defaults: {
		classes: 'shepherd-theme-arrows',
		scrollTo: true
	}
});
Explanation: You create a new tour variable. Then you initialize a new Shepherd tour object and save it to that tour variable. The defaults that you specify will be used across all of your Shepherd steps, unless overwritten later. The class 'shepherd-theme-arrow' is one of the default Shepherd themes we can apply.
